# Components Directory

This directory holds detailed component design documents, API specifications, and data model DDLs for the llm-reporting system. Content is added as components move from architecture design to detailed specification.

## Expected Contents (future)

| Subdirectory / File | Description |
|---------------------|-------------|
| `api/` | OpenAPI / Protobuf specifications for REST, gRPC, and MCP interfaces |
| `data-model/` | DDL scripts, migration files, schema evolution plans |
| `compute-spec/` | Detailed Compute Spec YAML schemas and validation rules |
| `agent-skills/` | Detailed Skill and MCP server specifications |
| `security/` | Detailed security policies, encryption specifications, compliance mappings |

## Current Status

The project is in the design phase. Component-level detailed specifications will be added as the architecture is refined and implementation approaches are finalized.

For the current architecture design, see:
- [docs/03-architecture.md](../docs/03-architecture.md) — Full architecture
- [adr/](../adr/) — Architecture Decision Records
